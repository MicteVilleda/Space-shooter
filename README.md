## 🛠️ Installation
1. Clone the repository: `git clone https://github.com/MicteVilleda/Space-shooter.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the game: `python code/main.py`

## 🎮 Controls
- **Move**: Arrow keys
- **Shoot:** Spacebar
